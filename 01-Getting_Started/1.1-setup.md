## Set up mlflow Environment
1. Create project folder
2. Open VS code and create conda environment
    ```
    conda create -p <environment-name> python==<python-version> -y
    ```
    Activate the environment
    ```
    conda activate <environment address>
    ```
3. Crate dependencies:
    - requirements.txt file and add `mlflow`

4. Install all dependencies
    ```
    pip install -r requirements.txt
    ```
5. Your setup is ready
